# BMoney - 全端小費用記帳系統 (MERN STACK)
## *- MVP - Work In Progress -*

本專案為第二專案，使用 RESTful API 架構，
實作完整 CRUD 功能，
並完成前後端分離與部署。

### 技術棧
- 前端: React.JS, TailwindCSS, React-Router
- 後端: Node.JS, Express, MongoDB

### Demo 連結
https://bmoney-sigma.vercel.app/records

### MVP 功能
- 新增記錄
- 顯示記錄
- 編輯記錄
- 刪除記錄
- 設定預算
- 顯示剩餘預算
- 顯示每月總支出與收入
- 顯示每日總支出與收入
